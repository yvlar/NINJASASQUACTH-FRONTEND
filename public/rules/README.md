# Règles de jeux (PDF)

Déposez ici les PDF de règles officiels, servis en statique par Vercel.
Les chemins publics attendus correspondent aux valeurs saisies dans l'admin
(`rules_pdf_fr` / `rules_pdf_en` d'un jeu), par exemple :

- `/rules/burgle-jack-regles-fr.pdf`
- `/rules/burgle-jack-rules-en.pdf`
- `/rules/heroes-rising-regles-fr.pdf`
- `/rules/heroes-rising-rules-en.pdf`

Tant qu'un fichier n'existe pas, laissez le champ correspondant **vide** dans
l'admin : la fiche affiche « Règles bientôt disponibles » plutôt qu'un lien
mort (aucun faux PDF n'est committé — contenu de marque fourni par l'éditeur).
